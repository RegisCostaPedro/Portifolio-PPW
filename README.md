## Funcionalidades

O projeto é simples ele é uma landing page de apresentação pessoal ele contém uma nav bar com links para certas páginas do site:
<ul>
  <li>Início</li>
  <li>Sobre</li>
  <li>Estudos</li>
  <li>Entrar em contato</li>
</ul>
