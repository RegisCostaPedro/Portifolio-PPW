Tecnólogias que utilizo

  • Java

    • Java é uma linguagem de programação versátil e robusta, reconhecida por sua portabilidade e orientação a objetos. Amplamente usada no desenvolvimento de aplicativos empresariais e web, destaca-se pela capacidade de execução em diversas plataformas.

  • Spring Boot

    • Um framework em Java que simplifica o desenvolvimento de aplicativos Java, facilitando a configuração e fornecendo um ambiente pronto para produção. É especialmente popular para o desenvolvimento de microserviços.

  • PostgreSQL

    • É um sistema de gerenciamento de banco de dados relacional de código aberto. É conhecido por sua velocidade e simplicidade, sendo uma escolha comum para aplicativos web e de pequeno a médio porte.

  • MySQL

    • É um sistema de gerenciamento de banco de dados relacional de código aberto, conhecido por sua confiabilidade e recursos avançados. É amplamente utilizado em aplicações que exigem manipulação eficiente de dados.
